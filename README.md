# superstore-sales-analysis
End-to-end Sales Analytics project using SQL, Python, DAX, and Power BI. Includes Executive Overview, ABC Product Analysis, Customer Insights, and Profitability Dashboard.

The goal is to identify:

- Sales performance
- Product profitability
- Customer behavior
- ABC product classification
- Revenue and profit drivers

---

## Tools Used

- SQL
- PostgreSQL
- Power BI
- DAX
